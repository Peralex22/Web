Programación

Existen muchísimos programas y lenguajes relacionados con la informática, estos son algunos de los tantos que he utilizado alguna vez. Cada vez se van actualizando y siempre hay alguna cosa que aprender, ya sean métodos más eficaces, más sencillos o nuevo material.

HTML

El HTML es el principal lenguaje de marcado utilizado por programadores y diseñadores web de cualquier nivel de experiencia

CSS

Dentro del diseño web se utiliza el lenguaje CSS , que sirve para definir y crear la presentación de una página web.

JavaScript

JavaScript es el lenguaje de programación por excelencia, ya que sirve para todo tipo de dispositivos y navegadores.

React

Es una librería JavaScript de código abierto, basada en componentes, para construir interfaces de usuario rápidas y dinámicas.

Bootstrap

Es un framework de desarrollo web gratuito y de código abierto. Está diseñado para facilitar el proceso de desarrollo de los sitios web responsivos y orientados a los dispositivos móviles,

Git

Git es un sistema de control de versiones distribuido gratuito y de código abierto diseñado para manejar todo, desde proyectos pequeños hasta proyectos muy grandes, con rapidez y eficiencia.
